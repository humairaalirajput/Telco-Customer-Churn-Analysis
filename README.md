# Telco Customer Churn Analysis

## Project Overview

This project analyzes customer churn using Python.
The goal of this project is to find the main reasons why customers leave the telecom company and identify factors affecting churn.

## Dataset

Telco Customer Churn Dataset
The dataset contains customer information such as:

* Gender
* Senior Citizen
* Contract Type
* Payment Method
* Tenure
* Monthly Charges
* Total Charges
* Churn

## Tools and Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Steps Performed

1. Data Loading
2. Data Cleaning
3. Data Type Conversion
4. Exploratory Data Analysis
5. Data Visualization
6. Churn Analysis
7. Summary and Recommendations

## Key Insights

* 26.54% customers have churned
* Month-to-month contract customers churn more
* Electronic check users have high churn
* Customers with low tenure churn more
* Senior citizens have higher churn rate
* Customers without extra services churn more

## Visualizations

* Count plots
* Pie chart
* Bar charts
* Stacked percentage charts
